# 📖 Como Usar SCAN

Guia completo para usar o SCAN - SFS Comprehensive Addon Network.

## Primeiros Passos

### 1. Instalação

1. Baixe a versão mais recente em [Releases](https://github.com/Del-SFS/SFS-Comprehesive-Addon-Network/releases)
2. Execute o instalador
3. Selecione a pasta de instalação do Spaceflight Simulator
4. Clique em "Instalar"

### 2. Primeira Execução

Ao abrir SCAN pela primeira vez:
- A interface carregará a lista de mods disponíveis
- O programa detectará automaticamente sua instalação do SFS
- Você verá um dashboard com estatísticas gerais

## Navegação Básica

### Abas Principais

#### 📦 Mods Disponíveis
- Veja todos os mods catalogados
- Filtre por categoria, autor ou compatibilidade
- Leia descrições e requisitos
- Veja avaliações e downloads

#### 🎯 Meus Mods
- Lista de mods instalados
- Versão atual de cada mod
- Opções para atualizar ou desinstalar
- Status de dependências

#### ⚙️ Configurações
- Pasta do SFS
- Preferências de atualização
- Temas (claro/escuro)
- Idioma

#### ℹ️ Sobre
- Versão do SCAN
- Créditos
- Links úteis

## Instalando Mods

### Passo a Passo

```
1. Vá para "Mods Disponíveis"
2. Procure pelo mod desejado (busca ou filtro)
3. Clique no mod para ver detalhes
4. Clique em "Instalar"
5. SCAN fará:
   ✓ Verificar dependências
   ✓ Detectar conflitos
   ✓ Baixar arquivo(s)
   ✓ Extrair na pasta correta
   ✓ Atualizar configurações
6. Reinicie o SFS
```

### Resolução de Dependências

Se um mod depende de outro:
- SCAN mostra um aviso
- Oferece instalar as dependências automaticamente
- Você pode revisar e confirmar
- Todos são instalados juntos

### Detecção de Conflitos

Se você tenta instalar um mod incompatível:
- SCAN mostra um ⚠️ **AVISO**
- Explica qual mod está em conflito
- Você pode:
  - Desinstalar o mod conflitante e continuar
  - Cancelar a instalação
  - Escolher qual manter

## Desinstalando Mods

### Remover um Mod

```
1. Vá para "Meus Mods"
2. Clique com botão direito no mod
3. Selecione "Desinstalar"
4. Confirme
```

### Limpeza de Dependências

Se um mod era apenas uma dependência e ninguém mais precisa dele:
- SCAN oferecerá remover automaticamente
- Você pode recusar se quiser manter

## Atualizando Mods

### Atualização Manual

```
1. Vá para "Meus Mods"
2. Procure por um ícone de atualização 🔄
3. Clique em "Atualizar"
4. Confirme
```

### Atualização Automática

Configure em **Configurações → Atualizações**:
- ✓ Verificar automaticamente
- ✓ Notificar quando houver atualizações
- ✓ Atualizar automaticamente (requer reinício)

## Buscando Mods

### Filtros Disponíveis

- **Categoria**: Gráficos, Física, Utilitários, etc.
- **Autor**: Encontre todos os mods de um criador
- **Compatibilidade**: Veja só mods compatíveis com sua versão
- **Classificação**: Mods mais bem avaliados
- **Downloads**: Mods mais populares
- **Recentes**: Lançamentos mais novos

### Busca por Texto

Digite na barra de pesquisa para encontrar:
- Nome do mod
- Descrição
- Tags

## Gerenciando Seu Perfil

### Estatísticas

No dashboard você vê:
- 📊 Total de mods instalados
- 📦 Espaço em disco usado
- 🆕 Atualizações disponíveis
- ⏱️ Última sincronização

### Histórico

- Veja todos os mods que já instalou
- Data de instalação/desinstalação
- Versões anteriores

## Solucionando Problemas

### Mod não instala

1. Verifique internet
2. Confirm espaço em disco
3. Reinicie SCAN
4. Se persistir, reporte em [Issues](https://github.com/Del-SFS/SFS-Comprehesive-Addon-Network/issues)

### Erro de compatibilidade

- Desinstale o mod conflitante primeiro
- Ou escolha atualizar para versão compatível

### Mod desapareceu

- Pode ter sido removido do DataCenter
- Verifique em "Histórico"
- Tente instalar uma versão mais antiga
